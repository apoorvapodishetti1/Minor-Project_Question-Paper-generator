Generating Automatic Question Paper
Getting papers may take so much time(sometimes), which can be a major hurdle for academic institutions, especially when they need to conduct exams on short notice.

Our project addresses this problem by introducing an automated question paper generation solution through a user-friendly application. 

Our application allows to generate question papers in just one click by choosing academic year, branch, subject, and marks, etc
Which allows us timely and efficiently respond to the needs of academic institutions.
The main motive is to generate question papers in a single click, for Academic institutions like Universities and Engineering colleges.

About the website:

•	Developed the project's frontend using ReactJS, ensuring an intuitive and user-friendly interface.

•	Integrated the frontend with the Flask framework for seamless communication with the database.

•	Utilized MySQL for efficient data storage, retrieval, and management within the project.
